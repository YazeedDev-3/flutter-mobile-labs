# Lab 3 — Flutter Registration Form

A simple registration form built with Flutter as part of a mobile development lab exercise.

---

## Screenshot

![App Screenshot](screenshot.png)

---

## Features

- Username and password input fields
- "Agree to terms of service" checkbox
- Register button that displays the entered credentials on submission
- Form only submits when the checkbox is checked
- Gradient background and modern card-based UI

---

## Getting Started

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install) installed
- A connected device or emulator

### Run the app

```bash
flutter pub get
flutter run
```
